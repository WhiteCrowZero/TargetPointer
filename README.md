# 寻音指针

“寻音指针”是一套基于上位机视觉识别与语音交互的桌面智能指向装置，面向课程设计与原型验证。项目采用“手机摄像头采集画面、电脑端完成识别与计算、Blue Pill 执行舵机控制”的最终方案，旧的独立外设识别与定位思路已废弃。

## 当前最终方案

系统链路固定为：

`手机摄像头 -> 电脑端（语音识别 + YOLO 目标检测 + 角度计算）-> 串口 -> Blue Pill -> SG90`

职责划分：

- 手机通过 USB 作为摄像头输入。
- 电脑端负责语音识别、目标类别匹配、YOLO 检测、目标方向计算和串口发令。
- Blue Pill 只负责接收串口命令并控制 SG90 转到指定角度。

## 技术路线

### 硬件

- 主控：`Blue Pill（STM32F103C8）`
- 执行机构：`SG90 微型舵机`
- 下载调试：`ST-LINK V2`
- 串口通信：`CH340 USB-TTL`
- 视频输入：`手机 USB 摄像头`
- 原型供电：`5V 电源 + HW-131 面包板电源模块`

### 软件

- 上位机语言：`建议 Python，最终实现待定`
- Python 环境管理：统一使用 `uv`
- 目标检测：`YOLO 系列模型，具体版本待选型`
- 视频处理：`建议 OpenCV`
- 串口通信：`建议 pyserial`
- 语音识别：`待选型，可先用桌面端现成语音输入方案`
- MCU 固件：`Blue Pill 串口接收 + SG90 控制`

## 快速开始

同步 Python 依赖：

```bash
uv sync
```

构建固件：

```bash
uv run pio run --project-dir firmware
```

串口联调：

```bash
uv run python scripts/pointer_serial_cli.py --port COM5 ping
uv run python scripts/pointer_serial_cli.py --port COM5 angle 120
```

运行视觉闭环：

```bash
uv run python scripts/pointer_vision_app.py --port COM5 --camera 0 --target cup --show
```

## 系统流程

1. 手机通过 USB 向电脑提供实时视频画面。
2. 用户在电脑端输入或说出目标名称，例如“水杯”。
3. 上位机将语音结果映射为目标类别。
4. YOLO 在当前视频帧中检测目标并得到目标框。
5. 上位机根据目标框中心点与画面中心的水平偏移计算目标方向。
6. 上位机将目标方向映射为舵机角度，并通过串口发送给 Blue Pill。
7. Blue Pill 控制 SG90 转动箭头，指向目标方向。

## 开发里程碑

1. Blue Pill 基础固件跑通，完成下载、串口和舵机控制。
2. 上位机完成串口发 `ANGLE:<deg>` 命令。
3. 手机通过 USB 稳定接入电脑并可读取视频流。
4. 上位机完成 YOLO 检测并输出目标框中心点。
5. 完成“目标中心点 -> 水平偏移 -> 舵机角度”映射。
6. 接入语音识别，形成“语音目标名 -> 检测类别 -> 舵机指向”完整链路。
7. 完成桌面场景演示和参数调优。

## 推荐调试顺序

1. 验证 Blue Pill + ST-LINK 下载。
2. 验证 SG90 安全转角与回中。
3. 验证 CH340 串口通信。
4. 用上位机串口工具发送角度命令控制舵机。
5. 验证手机 USB 摄像头接入。
6. 验证 YOLO 视频检测。
7. 验证角度映射与画面方向一致。
8. 完成整机联调。
