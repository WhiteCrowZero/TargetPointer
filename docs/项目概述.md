# 项目概述

## 项目背景

“寻音指针”是一套面向桌面近距离场景的智能指向原型，目标是在用户说出目标名称后，自动在画面中找到对应物体，并驱动实体箭头指向该目标的大致方向。项目重点是完成“语音输入 + 视觉识别 + 角度计算 + 物理指向”的完整链路，适合作为课程设计和原型演示。

## 项目目标

- 用手机作为外部摄像头采集桌面视频。
- 用电脑端完成语音识别和 YOLO 目标检测。
- 根据目标在画面中的水平位置计算指向角度。
- 通过串口把角度命令发送给 Blue Pill。
- 由 Blue Pill 控制 SG90 驱动箭头转向。

## 使用场景

- 室内桌面近距离演示。
- 摄像头位置固定，拍摄范围稳定。
- 目标物体数量有限，优先支持 2 到 3 类。
- 目标尽量无遮挡，便于稳定识别。

## 核心思路

本项目不在嵌入式端做视觉识别，也不在 MCU 上做语音处理。识别全部放在上位机完成，执行全部放在 MCU 完成。

- 识别在上位机：电脑端负责接收手机视频、处理语音输入、运行 YOLO、计算目标方向。
- 执行在 MCU：Blue Pill 只处理串口命令，并控制 SG90 转到指定角度。

这种分工可以明显降低 Blue Pill 的算力压力，缩短原型开发周期，也更适合桌面级课程设计项目。

## 简要工作流程

1. 手机通过 USB 向电脑提供实时视频。
2. 用户说出目标名称，例如“水杯”。
3. 上位机将语音结果映射为要查找的目标类别。
4. YOLO 在视频画面中找到目标并输出目标框。
5. 上位机根据目标框中心点相对画面中心的水平偏移计算目标方向。
6. 上位机将方向映射为 SG90 角度，通过串口发送给 Blue Pill。
7. Blue Pill 驱动舵机和箭头转到对应角度。

## 项目定位

“寻音指针”最终定位为：基于上位机视觉识别与语音交互的桌面智能指向装置。
