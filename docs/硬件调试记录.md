# 硬件调试记录

## 当前状态说明

本项目当前统一为“固定摄像头 + 上位机 YOLO 检测与控制 + Blue Pill + SG90”的单主线方案。旧的物体类别指向叙事和其他外围方案不再作为调试依据。

## 当前硬件盘点

### 已有硬件

- `Blue Pill`
- `STM32F103 核心板`
- `SG90`
- `ST-LINK V2`
- `CH340 USB-TTL`
- `HW-131`
- `面包板`
- `杜邦线`
- `电阻`
- `LED`

### 说明

- `Blue Pill` 是当前正式主控。
- `SG90` 是当前唯一执行机构。
- 当前结构默认服务于固定摄像头下的单轴水平指向。

## 待验证项

- 舵机独立 5V 供电稳定性
- 固定摄像头接入方式
- 上位机与 Blue Pill 串口联调稳定性
- 人物选择后的持续更新稳定性
- 水平偏移到舵机角度的实机标定参数

## 建议记录字段

- 日期
