# 硬件调试记录

## 当前状态说明

本项目已统一切换到“上位机视觉识别 + 语音交互 + Blue Pill 舵机执行”方案。旧的红外定位思路已废弃，不再作为后续调试依据。

## 当前硬件盘点

### 已有硬件

- `Blue Pill`
- `STM32F103 核心板`
- `ESP32（备用）`
- `SG90`
- `ST-LINK V2`
- `CH340 USB-TTL`
- `HW-131`
- `面包板`
- `杜邦线`
- `电阻`
- `LED`

### 说明

- `Blue Pill` 是当前正式主控。
- `STM32F103 核心板` 与 `ESP32` 仅作为备用板卡或对比实验资源，不作为当前主方案核心。
- `SG90` 是当前唯一执行机构。

## 已确认可用项

- `Blue Pill`
- `STM32F103 核心板`
- `ESP32（备用）`
- `SG90`
- `ST-LINK V2`
- `CH340 USB-TTL`
- `HW-131`
- `面包板 / 杜邦线 / 电阻 / LED`

## 待验证项

- 舵机供电稳定性
- 手机 USB 摄像头接入方式
- 上位机与 Blue Pill 串口联调
- 舵机安装结构的稳定性
- 目标中心点到舵机角度的标定参数
