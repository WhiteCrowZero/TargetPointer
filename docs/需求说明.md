# 需求说明

## 项目目标

“寻音指针”用于室内桌面场景。用户输入目标名称后，系统在视频画面中识别目标位置，并驱动实体箭头完成指向。

## 功能需求

1. 支持语音输入目标名称。
2. 上位机能够把语音结果映射为目标类别，例如“水杯”“钥匙”“遥控器”。
3. 上位机能够从手机摄像头视频流中读取实时画面。
4. YOLO 能够在视频画面中找到目标类别并输出目标框。
5. 上位机能够根据目标框中心点相对画面中心的水平位置计算目标方向。
6. 上位机能够把目标方向映射为舵机角度。
7. 上位机能够通过串口向 Blue Pill 发送角度命令。
8. Blue Pill 能够接收串口命令并驱动 SG90 转到指定角度。
9. 系统支持回中命令和停止命令，便于演示和调试。

## 非功能需求

- 演示场景限定为室内桌面近距离。
- 摄像头安装位置固定，不在运行中频繁移动。
- 目标尽量无遮挡，避免严重重叠。
- V1 先支持 2 到 3 类常见目标。
- 箭头只做单轴水平指向，不做俯仰控制。
- 识别与控制流程应便于调试和复现实验结果。
- Blue Pill 端保持简单，只处理串口与舵机控制。

## V1 范围内

- 手机 USB 摄像头视频输入
- 上位机语音输入和目标类别匹配
- 上位机 YOLO 目标检测
- 目标中心点到舵机角度的映射
- 串口文本协议控制 Blue Pill
- SG90 单轴水平指向

## V1 范围外

- 在 MCU 上运行视觉模型
- 在 Blue Pill 上直接做语音识别
- 双轴云台指向
- 室外远距离识别
- 连续跟踪高速移动目标

## 验收重点

- 用户能输入目标名称并触发一次完整指向流程。
- 上位机能稳定输出可用角度命令。
- Blue Pill 能在安全角度范围内驱动 SG90 指向目标方向。
- 整机可在固定桌面场景完成演示。
