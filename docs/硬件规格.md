# 硬件规格

本文件描述当前唯一硬件方案：固定摄像头 + 上位机运行 YOLO + Blue Pill + SG90 的单轴水平人物指向结构。

## 视觉输入

- 摄像头：`USB 摄像头` 或 `手机摄像头`
- 安装方式：固定机位
- 处理位置：上位机
- 当前用途：人物检测、目标初始化和后续位置更新

说明：

- 当前默认使用固定摄像头，不做云台随动。
- YOLO 在上位机运行，不在嵌入式端运行。
- 当前已经完成一轮原型验证，实际使用的是手机摄像头；独立固定 USB 摄像头仍是推荐的最终形态。

## 上位机

- 运行环境：PC 或笔记本
- 主要任务：摄像头读取、YOLO 推理、目标选择、角度计算、串口发送
- 推荐依赖：`OpenCV`、`ultralytics`、`pyserial`

## 主控

- 主控：`Blue Pill（STM32F103C8）`
- 逻辑电压：`3.3V`
- 下载方式：`ST-LINK V2（SWD）`
- 串口调试：`CH340 USB-TTL，115200 8N1`
- 板载指示：`PC13` 可作为目标阶段指示灯

说明：

- Blue Pill 只负责串口协议和 SG90 执行。
- 固件不处理目标语义、不运行检测模型。
- 当前固件支持按阶段控制板载 LED：目标已锁定或重获中亮起，丢失或空闲时熄灭。
- 当前固件上电后默认保持空闲，不自动 attach 舵机、不自动回中，等待显式控制命令。

## 执行机构

- 执行机构：`SG90`
- 控制方式：Blue Pill PWM
- 中心角：`90°`
- 安全角范围：`20° ~ 160°`
- 机械输出：轻质箭头或指针

说明：

- 当前只做单轴水平指向。
- 结构上应尽量减轻箭头重量，减少抖动。
- 固件侧对 `ANGLE` 和 `CENTER` 都采用渐进式到位，不直接大幅跳角。

## 调试与连接

- 下载器：`ST-LINK V2`
- 串口模块：`CH340 USB-TTL`
- 供电：`独立 5V 电源，共地`

供电约束：

- SG90 不从 Blue Pill 3.3V 引脚直接取电。
- 舵机使用独立 5V 更稳妥。
- 舵机地线、Blue Pill 地线、CH340 地线必须共地。

当前原型验证状态：

- 已完成短时功能验证时，舵机曾由 `CH340` 临时供电，核心功能可以跑通。
- 但这不是推荐的长期或最终提交硬件形态；若追求稳定性和安全余量，仍应切换到独立 5V 供电。

## 当前已有硬件

- `Blue Pill`
- `STM32F103 核心板`
- `SG90`
- `ST-LINK V2`
- `CH340 USB-TTL`
- `HW-131`
- `面包板`
- `杜邦线`
- `电阻 / LED`

## 建议补充

- 固定摄像头支架
- 稳定独立 5V 供电
- 箭头固定件
- 舵机安装底座

## 当前不包含

- 多轴云台结构
- 任何用于身份识别的专用硬件
- 与当前主线无关的额外输入前端
