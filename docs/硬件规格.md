# 硬件规格

本文件是“寻音指针”当前方案的单一事实来源。当前方案固定为“手机摄像头 + 上位机识别与控制 + Blue Pill + SG90”。

## 主控

- 主控：`Blue Pill（STM32F103C8）`
- 主频 / 逻辑电压：`72MHz / 3.3V`
- 下载方式：`ST-LINK V2（SWD）`
- 串口调试：`CH340 USB-TTL，115200 8N1`

说明：

- Blue Pill 只负责串口接收和 SG90 控制。
- 不在 Blue Pill 上运行视觉模型或语音识别。

## 语音前端

- 方案：`上位机软件实现`
- 输入位置：`电脑端`
- 实现状态：`待选型`

说明：

- 语音识别在上位机完成，输出目标名称或目标类别。
- 当前不使用独立语音外设。

## 视频与定位

- 定位方式：`手机摄像头 + YOLO 视觉检测`
- 摄像头来源：`手机通过 USB 接入电脑`
- 检测执行位置：`上位机`
- 输出结果：`目标框中心点与目标方向`

说明：

- 目标方向由上位机根据画面中的目标位置计算。
- 摄像头默认固定在桌面场景，不做移动跟踪平台。

## 执行机构

- 执行机构：`SG90`
- 控制方式：`Blue Pill PWM 控制`
- 回中角度：`90°`
- 安全转角范围：`20° ~ 160°`
- 供电电压：`5V`

说明：

- 箭头固定在 SG90 输出轴上。
- 当前只做单轴水平指向。

## 供电

- 系统供电：`5V`
- 原型电源模块：`HW-131`
- 舵机供电建议：`独立 5V 供电，共地`

供电约束：

- Blue Pill 使用 3.3V 逻辑。
- SG90 不直接从 Blue Pill 3.3V 引脚取电。
- 舵机电源与主控电源可以分开，但必须共地。

## 机械约束

- 结构形式：`轻质箭头 + 舵机支架`
- 工作场景：`室内桌面近距离`
- 摄像头位置：`固定`
- 目标状态：`尽量无遮挡`

## 已有硬件

- `Blue Pill`
- `STM32F103 核心板`
- `ESP32（备用）`
- `SG90`
- `ST-LINK V2`
- `CH340 USB-TTL`
- `HW-131`
- `面包板`
- `杜邦线`
- `电阻若干`
- `LED 若干`

## 仍需补充

- 手机固定支架
- 稳定 5V 电源
- 箭头固定件或轻质支架
- 舵机安装底座

## 已废弃方案

以下内容不再属于当前项目方案：

- 独立语音外设
- 视觉外的本地定位方案
- 旧定位附件及相关接收器件
- 依赖外设定位的旧流程
